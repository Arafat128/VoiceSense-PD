VoiceSense Research

Voice-based Parkinson’s screening

Explainable machine learning for English & Bangla voice patterns. Built for thesis research — screening support only, never a clinical diagnosis.

Live voice studio

Record or upload sustained vowels. Clinical acoustic features + conservative thresholds.

Feature lab

UCI English & BenSParX Bengali feature-table models with detailed explanations.

Responsible design

Subject-aware training, quality gates, and Uncertain band to reduce false PD flags.

View GitHub repo How to run the full app
Why not full ML on Vercel? The screening engine uses Streamlit + scikit-learn + librosa + Praat (parselmouth) and multi‑MB models. Vercel serverless is not suitable for that stack. This landing page is hosted on Vercel; run the full interactive app locally (or Streamlit Cloud) from the GitHub repository:

git clone https://github.com/Arafat128/voicesense-pd-screening.git
cd voicesense-pd-screening
pip install -r requirements.txt
python -m streamlit run app/streamlit_app.py
